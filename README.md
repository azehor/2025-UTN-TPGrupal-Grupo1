# TP Grupal Desarrollo de Software 2025 - Grupo 1

- **frontend** → React + Vite  
- **api** → Go

## Crear archivo .env en /api
```bash
# Postgres
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_SSLMODE=

# puerto
APP_PORT=

# (development/production)
APP_ENV=development

````

## Frontend
```bash
cd frontend/
npm install 
npm run dev
````

## API

```bash
cd api/
go run .
```
